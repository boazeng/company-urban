# 🎼 לוח הזמנים של החברה

מקור האמת לתזמון. **המנצח זובין** ([[conductor]]) קורא את הטבלה הזו בכל פעימה ומפעיל את הסוכנים שהגיע זמנם.

- **טריגר `מתוזמן`** — זובין מפעיל לפי השעה שבטבלה.
- **טריגר `זמן-אמת`** — זובין **לא** נוגע; הסוכן רץ באופן עצמאי מ-webhook/אירוע חיצוני.
- זובין מפעיל רק שורות בסטטוס **`פעיל`**.

## הסבר שדות
- **מתי** — שעה בפורמט `HH:MM` (התאמה לפי שעה מול הפעימה). ריק עבור `זמן-אמת`.
- **ימים** — `יומי` / יום בשבוע (`ראשון`...) / רשימה מופרדת בפסיקים.
- **סטטוס** — `פעיל` (זובין מפעיל) / `מתוכנן` / `כבוי`.
