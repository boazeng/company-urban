# 🎼 המנצח זובין — סוכן התזמון

**פקודה:** `/conductor`
**סטטוס:** ✅ פעיל (מקומי)
**הגדרה:** [[conductor]] (`.claude/commands/conductor.md`)
**לוח הזמנים:** [[Schedule]]

## מה הוא עושה
זובין הוא "מנצח התזמורת" של החברה. בכל פעימה הוא קורא את [[Schedule]], בודק מי מהסוכנים המתוזמנים הגיע זמנו, ומפעיל אותם — למעט סוכני **זמן-אמת** שרצים עצמאית מ-webhook. מונע הרצה כפולה דרך יומן ריצה יומי, ורושם כל הפעלה.

## איך זה רץ — "פעימה" אחת בלבד
התשתית מפעילה רק את זובין על פי פעימה קבועה; זובין מחליט את כל השאר.

**מקומית (Windows Task Scheduler) — לבדיקה:**
```powershell
schtasks /create /tn "ConductorZubin" /sc hourly /tr "cmd /c cd /d \"c:\Users\User\Aiprojects\obsi_comp\" && claude -p \"/conductor\"" /f
```

**בשרת (cron) — כל שעה:**
```
0 * * * * cd /path/to/obsi_comp && claude -p "/conductor" >> conductor.cron.log 2>&1
```

> פעימה כל שעה מתאימה ללוח שעות עגולות. רוצים רזולוציה דקות? הריצו כל 15 דק' והתאימו את התאמת השעה בשלב 3 לחלון פעימה.

## שיחה במערכת התקשורת (Comms)
זובין הוא גם **משתתף בחדרים** (מוח bespoke: `zubin_core.py`), עם שני מצבים:
- **שיחה (ברירת מחדל)** — עונה על לוח הזמנים מתוך [[Schedule]] (מי מתוזמן מתי, ניתוח, המלצות שינוי). לא מפעיל כלום.
- **פעימה לפי בקשה** — בקשה מפורשת ("הרץ פעימה" / "רוץ עכשיו" / "תפעיל את התור") מריצה את `/conductor` האמיתי.

> זיהוי הפעימה שמרני בכוונה — שאלות (\"מי מתוזמן מחר?\") נשארות שיחה ולא מפעילות סוכנים.
