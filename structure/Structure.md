# 🏢 מבנה החברה

מקור האמת למבנה הארגוני. עמוד "מבנה חברה" באתר קורא את הטבלה הזו וחי.

- **סוכן** — מפתח הזהות (משמש את עמודת "כפוף ל"). עבור צמתים בעלי שם פרטי זהו השם; עבור צמתי הנהלה ללא שם פרטי זהו התפקיד.
- **תפקיד** — התפקיד המוצג מודגש בכרטיס.
- **שם** — שם פרטי של הסוכן (מוצג בסוגריים מתחת לתפקיד). ריק עבור צמתים שאין להם שם פרטי (תפקידי הנהלה גנריים).
- **רמה** — הרמה של הסוכן בהיררכיית החברה. ההנהלה = רמה `1`. רמות נוספות (`2`, `3`...) נוספות ככל שהארגון גדל. ניתן לשנות את הרמה של כל סוכן כאן.
- **כפוף ל** — שם הסוכן שמעליו (לציור התרשים). `—` עבור הצומת העליון.
- **מטרה** — ייעוד הסוכן (אופציונלי; ריק עבור ההנהלה).
- **קישור** — קישור לכלי של הסוכן (אופציונלי; מוצג ככפתור בכרטיס). לגיא — עורך התסריט (`http://localhost:5210`, רץ מקומית אך מחובר ל-DynamoDB החי).
- **סוג** — `הנהלה` (קו פיקוד) / `סוכן` / `עוזר` (עוזר אישי — מוצג מובחן, לצד המנהל ולא בשורת הכפיפים).
